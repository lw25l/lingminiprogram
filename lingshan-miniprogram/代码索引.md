## 💡 核心算法说明

### 1. 智能问答算法
```javascript
// 三层问答策略
1. 直接匹配 → 精确答案
2. 模糊匹配 → 相关答案
3. 关键词匹配 → 推荐景点
4. 默认回复 → 引导提问
```

### 2. 推荐算法
```javascript
// 基于兴趣标签匹配
兴趣标签 → 匹配景点标签 → 推荐相关景点
佛教文化 → 灵山大佛、五印坛城
历史人文 → 灵山梵宫、大照壁
自然风光 → 菩提大道
```

### 3. 路线规划算法
```javascript
// 多条件筛选
时长 + 人群 + 难度 → 推荐路线
短时长 + 家庭 + 轻松 → 亲子休闲游
长时长 + 情侣 + 中等 → 深度文化游
```
