# 🚀 小程序运行指南

## ⚠️ 重要说明

微信小程序**必须在微信开发者工具中运行**，不能在浏览器或Node.js环境中直接运行。

---

## 📱 如何运行小程序

### 方法1：使用微信开发者工具（推荐）

#### 步骤1：下载并安装微信开发者工具
- 下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 选择适合您系统的版本（Windows/Mac/Linux）

#### 步骤2：克隆项目
```bash
git clone https://github.com/lw25l/lingminiprogram.git
cd lingminiprogram
```

#### 步骤3：打开项目
1. 启动微信开发者工具
2. 选择"小程序项目"
3. 目录选择：`lingminiprogram/miniprogram`
4. AppID：使用测试号（点击"使用测试号"）或输入您的AppID
5. 点击"确定"

#### 步骤4：准备图片资源（可选）
项目已包含所有代码，但需要图片资源才能完整显示：

1. 进入 `miniprogram/assets/images/` 目录
2. 参考 `README.md` 添加景点图片
3. 或暂时使用占位图测试

**临时方案**：可以先不添加图片，小程序仍可运行，只是图片区域为空。

#### 步骤5：编译运行
- 点击工具栏的"编译"按钮
- 在模拟器中查看小程序运行效果

## 🔧 后端服务（可选）

如果需要完整的后端功能：

```bash
# 进入服务端目录
cd lingminiprogram/server

# 安装依赖
npm install

# 启动服务
npm start
```

服务将在 `http://localhost:3000` 运行

## ❓ 常见问题

### Q: 为什么不能直接运行？
A: 微信小程序是微信平台的应用，必须在微信开发者工具中运行。

### Q: 没有图片资源怎么办？
A: 可以先运行测试功能，图片只是视觉展示，不影响核心功能。

### Q: 如何获取微信AppID？
A:
1. 注册微信小程序：https://mp.weixin.qq.com
2. 在开发设置中获取AppID
3. 或直接使用测试号

### Q: 后端服务必须吗？
A: 不是必须的。小程序已包含本地知识库，可以独立运行。后端用于：
- 接入AI大模型
- 用户数据存储
- 更多高级功能
