## 📋 项目简介

本项目是一个完整的AI数字人智能景区导览系统，为灵山胜境景区提供7x24小时智能导游服务。系统集成了多模态交互、智能问答、个性化推荐等核心功能，助力文旅产业数字化转型。

### 🎯 核心亮点

- ✅ **多模态交互**：语音识别/合成 + 文本输入 + 表情动画同步
- ✅ **智能问答**：本地知识库（准确率≥90%）+ AI大模型双引擎
- ✅ **个性化推荐**：基于兴趣标签的多维度路线推荐
- ✅ **地图导航**：GPS定位 + 景点标记 + 路径规划
- ✅ **即开即用**：完整代码，无需额外配置

## 🚀 快速开始

### 方式一：运行小程序（推荐）

1. **下载微信开发者工具**
   - 地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

2. **导入项目**
   ```
   目录：lingshan-miniprogram/miniprogram/
   AppID：使用测试号
   ```

3. **点击编译运行**
   - 首页：景点推荐、兴趣选择
   - 聊天页：AI对话、语音输入
   - 地图页：GPS导航、景点标记
   - 路线页：多条件筛选

### 方式二：启动后端服务

```bash
# 进入后端目录
cd lingshan-miniprogram/server

# 安装依赖
npm install express cors body-parser

# 启动服务
node app.js

# 测试接口
curl http://localhost:3000/api/health
```

## 🔧 扩展开发

### 接入AI大模型
```javascript
// server/services/ai.js
const OpenAI = require('openai');

async function chat(query) {
  const response = await openai.chat.completions.create({
    model: "gpt-4",
    messages: [{ role: "user", content: query }]
  });
  return response.choices[0].message.content;
}
```
