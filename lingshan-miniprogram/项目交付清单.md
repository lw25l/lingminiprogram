## 🚀 快速启动步骤

### 1. 创建项目结构
```bash
mkdir -p lingshan-miniprogram/{miniprogram,server}
cd lingshan-miniprogram
```

### 2. 复制代码文件
将展示的所有代码复制到对应文件中。

### 3. 安装后端依赖
```bash
cd server
npm install
```

### 4. 启动后端服务
```bash
npm start
```

### 5. 导入小程序项目
- 打开微信开发者工具
- 选择 `miniprogram` 目录
- 填写AppID（或使用测试号）

### 6. 准备资源文件
将图片和图标放入 `assets` 目录。

### 7. 开始调试
在开发者工具中预览和调试。
