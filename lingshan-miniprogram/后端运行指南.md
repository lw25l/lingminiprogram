# 🔧 后端服务运行指南

## 📦 后端架构

项目包含完整的Node.js + Express后端服务！

### 后端技术栈
- **运行环境**: Node.js
- **Web框架**: Express
- **数据库**: MongoDB（可选，默认使用JSON文件）
- **AI服务**: 可接入讯飞星火大模型

---

## 🚀 快速启动后端

### 第一步：安装Node.js

#### Windows/Mac
下载地址：https://nodejs.org/
选择 LTS 版本下载安装

#### Linux
```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs
```

验证安装：
```bash
node --version  # 应显示 v18.x.x 或更高
npm --version   # 应显示 9.x.x 或更高
```

---

### 第二步：安装依赖

```bash
# 进入后端目录
cd lingminiprogram/server

# 安装依赖包
npm install
```

安装的依赖包括：
- express（Web框架）
- cors（跨域支持）
- body-parser（请求解析）
- multer（文件上传）
- mongoose（MongoDB，可选）

---

### 第三步：启动服务

```bash
# 方式1：直接启动
npm start

# 方式2：开发模式（自动重启）
npm run dev
```

启动成功会显示：
```
🚀 灵山胜境AI导览服务已启动
📡 服务地址: http://localhost:3000
⏰ 启动时间: 2026-05-03 22:56:00
```

## 🔗 前后端连接

### 修改前端API地址

编辑 `miniprogram/app.js`：

```javascript
globalData: {
  // 本地开发（连接本地后端）
  apiBaseUrl: 'http://localhost:3000/api',

  // 生产环境（连接线上后端）
  // apiBaseUrl: 'https://api.lingshan-guide.com/api',
}
```

### 配置微信开发者工具

1. 打开微信开发者工具
2. 右上角 **"详情"** → **"本地设置"**
3. 勾选 **"不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书"**

这样才能访问本地 `http://localhost:3000`

## 🗄️ 数据库配置（可选）

### 使用MongoDB

#### 1. 安装MongoDB
- Windows/Mac: https://www.mongodb.com/try/download/community
- Linux: `sudo apt install mongodb`

#### 2. 创建配置文件
在 `server/` 目录创建 `.env` 文件：
```env
MONGODB_URI=mongodb://localhost:27017/lingshan
PORT=3000
```

#### 3. 启动MongoDB
```bash
# Windows
net start MongoDB

# Mac/Linux
sudo systemctl start mongod

# 或使用Docker
docker run -d -p 27017:27017 --name mongodb mongo:latest
```

#### 4. 连接数据库
后端会自动连接MongoDB并创建数据表。

## 🤖 接入AI大模型（可选）

### 接入讯飞星火大模型

#### 1. 注册讯飞开放平台
访问：https://www.xfyun.cn/

#### 2. 创建应用
- 选择"星火认知大模型"
- 获取 AppID、APIKey、APISecret

#### 3. 配置环境变量
在 `server/` 目录创建 `.env` 文件：
```env
# 讯飞星火配置
XUNFEI_APP_ID=你的AppID
XUNFEI_API_KEY=你的APIKey
XUNFEI_API_SECRET=你的APISecret
AI_API_KEY=你的APIKey
```

#### 4. 安装SDK
```bash
npm install spark-node-sdk
```

#### 5. 修改AI服务
编辑 `server/services/ai.js`，接入讯飞SDK。

## 🚀 部署到云服务器

### 部署步骤

#### 1. 购买云服务器
- 阿里云、腾讯云、华为云等
- 推荐：2核4G，Ubuntu 20.04

#### 2. 安装环境
```bash
# 安装Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 安装PM2（进程管理）
sudo npm install -g pm2

# 安装MongoDB（可选）
sudo apt install mongodb
```

#### 3. 上传代码
```bash
# 使用Git
git clone https://github.com/lw25l/lingminiprogram.git
cd lingminiprogram/server
npm install
```

#### 4. 启动服务
```bash
# 使用PM2启动
pm2 start app.js --name lingshan-api

# 查看状态
pm2 status

# 查看日志
pm2 logs lingshan-api
```

#### 5. 配置Nginx反向代理
```nginx
server {
    listen 80;
    server_name api.lingshan-guide.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

#### 6. 配置HTTPS（推荐）
使用Let's Encrypt免费证书：
```bash
sudo apt install certbot python3-certbot-nginx
sudo certbot --nginx -d api.lingshan-guide.com
```

## 🔍 故障排查

### 后端无法启动？
```bash
# 检查端口占用
lsof -i :3000

# 杀死占用进程
kill -9 <PID>

# 或更换端口
PORT=3001 npm start
```

### API返回错误？
```bash
# 查看详细日志
DEBUG=* npm start

# 检查请求格式
# 确保Content-Type: application/json
```

### 无法连接MongoDB？
```bash
# 检查MongoDB状态
sudo systemctl status mongod

# 启动MongoDB
sudo systemctl start mongod
```
