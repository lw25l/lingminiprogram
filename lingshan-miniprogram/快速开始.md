# 灵山胜境AI智能导游小程序 - 快速开始指南

## 🚀 5分钟快速启动

### 第一步：下载项目

**方式1：克隆Git仓库**
```bash
git clone https://github.com/your-username/lingshan-miniprogram.git
```

**方式2：下载ZIP压缩包**
- 点击项目页面右上角"Code" → "Download ZIP"

---

### 第二步：导入微信开发者工具

1. 打开**微信开发者工具**
2. 选择**小程序项目**
3. 目录选择：`lingshan-miniprogram/miniprogram`
4. AppID：使用测试号或输入您的AppID
5. 点击**确定**

---

### 第三步：预览小程序

- 点击**编译**按钮
- 在模拟器中预览效果
- 测试各页面功能

## 🌐 后端服务（可选）

### 本地启动后端

```bash
# 进入服务端目录
cd lingshan-miniprogram/server

# 安装依赖
npm install

# 启动服务
npm start
```

服务将在 `http://localhost:3000` 运行

### 修改API地址

```javascript
// miniprogram/app.js 第6行
apiBaseUrl: 'http://localhost:3000/api'  // 本地开发
```

## 📖 常见问题

### Q: 编译报错"文件未找到"？
A: 确保所有文件都已正确创建，特别是图片资源文件

### Q: 地图不显示？
A: 需要在微信公众平台配置地图权限和域名白名单

### Q: 语音识别不工作？
A: 需要接入讯飞语音识别API或微信同声传译插件

### Q: 如何接入真实AI？
A: 修改 `server/services/ai.js`，接入讯飞星火或ChatGPT API
