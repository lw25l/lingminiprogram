# 🚀 如何在您的电脑上运行小程序

## 📋 准备工作

### 需要准备的工具：
1. **微信开发者工具**（必须）
2. **Git**（可选，用于克隆项目）

---

## 第一步：下载微信开发者工具

### 1.1 下载地址
访问官网：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 1.2 选择版本
- **Windows**: 下载 `.exe` 安装包
- **Mac**: 下载 `.dmg` 安装包
- **Linux**: 下载 `.tar.gz` 压缩包

### 1.3 安装
- Windows: 双击安装包，一路"下一步"
- Mac: 双击 `.dmg`，拖到应用程序文件夹
- Linux: 解压后运行

---

## 第二步：下载项目代码

### 方法1：使用Git（推荐）

#### 2.1 安装Git（如果没有）
- **Windows**: https://git-scm.com/download/win
- **Mac**: 终端运行 `xcode-select --install`
- **Linux**: `sudo apt install git`

#### 2.2 克隆项目
打开命令行/终端，运行：

```bash
# 克隆项目
git clone https://github.com/lw25l/lingminiprogram.git

# 进入项目目录
cd lingminiprogram
```

### 方法2：直接下载ZIP

1. 访问：https://github.com/lw25l/lingminiprogram
2. 点击绿色按钮 "Code"
3. 选择 "Download ZIP"
4. 解压到任意文件夹

---

## 第三步：在微信开发者工具中打开

### 3.1 启动微信开发者工具
- 双击桌面图标启动
- 使用微信扫码登录

### 3.2 创建项目

#### 方式A：使用测试号（最简单）

1. 点击 **"+"** 或 **"导入项目"**
2. 填写信息：
   - **项目名称**: 灵山胜境AI导游
   - **目录**: 点击"选择"，找到您刚才下载的 `lingminiprogram/miniprogram` 文件夹
   - **AppID**: 点击 **"使用测试号"** 按钮
3. 点击 **"确定"**

#### 方式B：使用自己的AppID

1. 如果您有微信小程序AppID，直接输入
2. 没有的话，访问 https://mp.weixin.qq.com 注册小程序

### 3.3 等待编译
- 工具会自动编译项目
- 几秒钟后，模拟器会显示小程序界面

---

## 第四步：运行测试

### 4.1 查看首页
- 模拟器中会显示小程序首页
- 可以看到轮播图区域、快捷入口等

### 4.2 测试AI导游

1. 点击底部Tab **"智能导游"**
2. 在输入框输入问题：
   - **"灵山大佛有多高？"**
   - **"推荐一条游览路线"**
   - **"门票多少钱？"**
   - **"九龙灌浴几点开始？"**
3. 点击发送，查看AI回复

### 4.3 测试其他功能
- **导览地图**: 查看景点标记
- **路线推荐**: 浏览预设路线
- **景点详情**: 点击景点查看详情

## ⚡ 常见问题

### Q1: 提示"文件未找到"？
**A**: 确保选择的目录是 `lingminiprogram/miniprogram`，不是 `lingminiprogram`

### Q2: 图片不显示？
**A**: 正常现象，需要添加图片资源。不影响功能测试。

### Q3: 如何在真机测试？
**A**:
1. 点击工具栏"预览"按钮
2. 用微信扫描二维码
3. 在手机上打开小程序

### Q4: 提示AppID错误？
**A**: 点击"使用测试号"即可，无需真实AppID

### Q5: 编译报错？
**A**:
1. 检查网络连接
2. 点击"清缓存" → "全部清除"
3. 重新点击"编译"
