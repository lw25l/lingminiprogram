# 灵山胜境AI数字人智能导览系统 - 项目文件清单

## 📦 项目概述

本项目是一个完整的AI数字人智能景区导览系统，包含微信小程序前端和Node.js后端服务。

**文件总数**: 40个核心文件
**开发框架**: 微信小程序原生 + Node.js/Express
**知识库**: 9个景点详细数据 + 4条预设路线

---

## 📁 文件结构

### 一、小程序前端 (miniprogram/)

#### 1. 全局配置
- `app.js` - 应用入口，全局数据管理
- `app.json` - 应用配置，页面路由、底部导航
- `app.wxss` - 全局样式，主题色变量

#### 2. 首页模块 (pages/index/)
- `index.wxml` - 首页布局（Banner、快捷入口、景点推荐）
- `index.wxss` - 首页样式
- `index.js` - 首页逻辑（兴趣选择、推荐算法）
- `index.json` - 页面配置

#### 3. AI聊天模块 (pages/chat/)
- `chat.wxml` - 聊天界面（数字人、消息列表、输入框）
- `chat.wxss` - 聊天样式（语音波形、动画效果）
- `chat.js` - 聊天逻辑（语音识别、AI对话、知识库查询）
- `chat.json` - 页面配置

#### 4. 地图导航模块 (pages/map/)
- `map.wxml` - 地图界面（景点标记、筛选标签）
- `map.wxss` - 地图样式
- `map.js` - 地图逻辑（GPS定位、路径规划）
- `map.json` - 页面配置

#### 5. 路线推荐模块 (pages/route/)
- `route.wxml` - 路线列表（筛选条件、路线卡片）
- `route.wxss` - 路线样式
- `route.js` - 路线逻辑（多条件筛选、推荐算法）
- `route.json` - 页面配置
- `detail.wxml` - 路线详情页
- `detail.wxss` - 路线详情样式
- `detail.js` - 路线详情逻辑

#### 6. 景点详情模块 (pages/attraction/)
- `attraction.wxml` - 景点详情（轮播图、介绍、文化内涵）
- `attraction.wxss` - 景点详情样式
- `attraction.js` - 景点详情逻辑
- `attraction.json` - 页面配置

#### 7. 个人中心模块 (pages/profile/)
- `profile.wxml` - 个人中心（数据统计、功能菜单）
- `profile.wxss` - 个人中心样式
- `profile.js` - 个人中心逻辑
- `profile.json` - 页面配置

#### 8. 工具函数 (utils/)
- `api.js` - API请求封装（GET/POST/上传）
- `knowledge.js` - 本地知识库（景点数据、问答匹配）

---

### 二、后端服务 (server/)

#### 1. 服务入口
- `app.js` - Express服务器启动、路由配置

#### 2. API路由 (api/)
- `chat.js` - 聊天接口（语音识别/合成、AI对话）
- `attractions.js` - 景点接口（列表、详情、搜索、推荐）
- `routes.js` - 路线接口（预设路线、个性化推荐）

#### 3. 知识库 (knowledge/)
- `attractions.js` - 景点完整数据（9个景点详细信息）

## 🚀 使用方法

### 方法一：直接查看代码
所有文件已保存在：
```
/root/.openclaw/workspace/lingshan-miniprogram/
```

### 方法二：微信开发者工具
1. 打开微信开发者工具
2. 导入项目：选择 `miniprogram` 目录
3. 填写AppID（可使用测试号）
4. 点击编译运行

### 方法三：启动后端服务
```bash
cd server
npm install express cors body-parser
node app.js
# 访问 http://localhost:3000/api/health
```
