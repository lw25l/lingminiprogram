## 第二步：安装开发工具

### 1. 微信开发者工具
- 下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 安装后登录微信账号

### 2. Node.js 环境
- 下载地址：https://nodejs.org/
- 推荐版本：LTS v18+ 或 v20+

---

## 第三步：创建小程序项目

### 1. 打开微信开发者工具
- 选择"小程序项目"
- 项目目录：选择 `lingshan-miniprogram/miniprogram` 文件夹
- AppID：使用测试号或填写您的AppID
- 项目名称：灵山胜境AI智能导览

### 2. 复制核心文件
将我之前展示的代码复制到对应文件中：

**必选文件（按顺序）：**
1. `app.json` - 小程序配置
2. `app.js` - 应用入口
3. `app.wxss` - 全局样式
4. `utils/knowledge.js` - 知识库（核心）
5. `utils/api.js` - API封装
6. `pages/index/index.*` - 首页（4个文件）
7. `pages/chat/chat.*` - AI聊天页（4个文件）

---

## 第四步：启动后端服务

### 1. 初始化后端项目
```bash
cd lingshan-miniprogram/server
npm init -y
```

### 2. 安装依赖
```bash
npm install express cors body-parser multer mongoose
```

### 3. 创建 package.json
```json
{
  "name": "lingshan-server",
  "version": "1.0.0",
  "description": "灵山胜境AI导览后端服务",
  "main": "app.js",
  "scripts": {
    "start": "node app.js",
    "dev": "nodemon app.js"
  },
  "dependencies": {
    "express": "^4.18.2",
    "cors": "^2.8.5",
    "body-parser": "^1.20.2",
    "multer": "^1.4.5-lts.1",
    "mongoose": "^7.0.0"
  },
  "devDependencies": {
    "nodemon": "^2.0.22"
  }
}
```

### 4. 启动服务
```bash
npm start
```

服务将在 http://localhost:3000 启动

## 🔧 开发注意事项

### 1. 真机调试
某些功能需要在真机上测试：
- 语音识别
- GPS定位
- 地图显示
- 摄像头调用

### 2. 域名配置
在微信公众平台配置合法域名：
- request域名：`https://api.lingshan-guide.com`
- uploadFile域名：`https://api.lingshan-guide.com`
- downloadFile域名：`https://api.lingshan-guide.com`

### 3. 权限配置
在 `app.json` 中已配置：
```json
{
  "permission": {
    "scope.userLocation": {
      "desc": "您的位置信息将用于为您提供导航服务"
    }
  },
  "requiredPrivateInfos": ["getLocation", "chooseLocation"]
}
```
