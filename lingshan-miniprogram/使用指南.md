## 七、如何接入AI大模型？

### 方法1: 使用OpenAI
```javascript
// server/services/ai.js
const OpenAI = require('openai');

const openai = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY
});

async function chat(query) {
  const response = await openai.chat.completions.create({
    model: "gpt-4",
    messages: [{ role: "user", content: query }]
  });
  return response.choices[0].message.content;
}
```

### 方法2: 使用通义千问
```javascript
const Qwen = require('@alicloud/qwen');

const client = new Qwen({
  apiKey: process.env.QWEN_API_KEY
});

async function chat(query) {
  const response = await client.chat({
    model: "qwen-max",
    messages: [{ role: "user", content: query }]
  });
  return response.output.text;
}
```

### 方法3: 使用讯飞星火
```javascript
const Spark = require('spark-ai');

const spark = new Spark({
  appId: process.env.SPARK_APP_ID,
  apiKey: process.env.SPARK_API_KEY
});

async function chat(query) {
  const response = await spark.chat(query);
  return response.content;
}
```

## 八、常见问题

### Q1: 小程序无法加载图片？
**A**: 需要在 `miniprogram/assets/images/` 目录下添加图片文件，或使用网络图片URL。

### Q2: 语音识别不工作？
**A**: 需要在 `app.json` 中添加权限：
```json
{
  "permission": {
    "scope.record": {
      "desc": "用于语音输入"
    }
  }
}
```

### Q3: 地图不显示？
**A**: 需要在微信公众平台开通「地图」权限，并配置域名白名单。

### Q4: 如何部署到生产环境？
**A**: 
1. 后端部署到服务器（阿里云/腾讯云）
2. 配置HTTPS域名
3. 修改 `api.js` 中的 `BASE_URL`
4. 提交小程序审核
